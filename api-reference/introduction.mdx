## Autenticação

Todas as APIs utilizam autenticação por meio de Bearer tokens, que devem ser informados no cabeçalho de cada requisição. Para obter seu token de autenticação, acesse a página [**Chave de API**](https://app.gptmaker.ai/browse/developers)
